# Francesco Marcucci — Portfolio

Portfolio personale di Francesco Marcucci, web developer in crescita e sport photographer.

## Tecnologie

- HTML5, CSS3, JavaScript (vanilla)
- Font Awesome 6 (icone)
- Google Fonts (Space Grotesk, Manrope)
- Tema chiaro/scuro con preferenza di sistema e salvataggio locale
- Intersection Observer per animazioni scroll-based

## Struttura

- `index.html` — pagina principale
- `style.css` — stili globali, tema scuro, responsive
- `Logo.png` — logo / avatar

## Sezioni

- **Hero** — presentazione e focus attuale

- **Progetti** — portfolio lavori (clienti + repo GitHub)
- **Fotografia** — galleria sportiva con link al sito dedicato
- **Contatti** — email e GitHub
